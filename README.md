# 📊 Data Analysis Projects

A curated collection of end-to-end data analysis projects covering sales intelligence, personal finance, HR analytics, and sports analytics — built with Python, SQL, and Tableau.

---

## 📁 Project Structure

```
Data-Analysis-Projects/
│
├── 1_SalesInsights/           # Sales performance analysis using Python & SQL
├── 2_SalesInsightsTableau/    # Tableau dashboard for sales insights
├── 3_PersonalFinance/         # Personal finance tracker & expense analysis
├── 4_HRAnalytics/             # HR attrition and workforce analytics
└── 5_CricketT20Analytics/     # T20 cricket match & player performance analysis
```

---

## 🚀 Projects Overview

### 1. 🛒 Sales Insights
End-to-end sales data analysis pipeline. Explores revenue trends, regional performance, and product-level breakdowns using Python and SQL queries on transactional data.

- **Tools:** Python, Pandas, SQL, Matplotlib
- **Key Skills:** Data cleaning, EDA, KPI analysis

---

### 2. 📈 Sales Insights (Tableau)
An interactive Tableau dashboard built on top of the Sales Insights dataset, enabling business stakeholders to explore KPIs through visualizations.

- **Tools:** Tableau
- **Key Skills:** Dashboard design, data storytelling, calculated fields

---

### 3. 💰 Personal Finance
Analysis of personal income and expense data to uncover spending patterns, savings rate, and financial health indicators.

- **Tools:** Python, Pandas, Matplotlib, Seaborn
- **Key Skills:** Budget analysis, time-series trends, category-level insights

---

### 4. 👥 HR Analytics
Workforce analytics project focused on employee attrition prediction and HR KPIs such as satisfaction, tenure, and department-level churn.

- **Tools:** Python, Pandas, Scikit-learn, Seaborn
- **Key Skills:** Attrition modeling, feature analysis, HR dashboards

---

### 5. 🏏 Cricket T20 Analytics
Performance analysis of T20 cricket matches and players, including batting/bowling statistics, team comparisons, and match outcome trends.

- **Tools:** Python, Pandas, Matplotlib, Seaborn
- **Key Skills:** Sports analytics, statistical summaries, player ranking

## ⚙️ Getting Started

### Prerequisites
Make sure you have the following installed:
- Python 3.8+
- Jupyter Notebook or JupyterLab
- Required libraries (see below)

### Installation

```bash
# Clone the repository
git clone https://github.com/Musawir456/Data-Analysis-Projects.git

# Navigate into the project folder
cd Data-Analysis-Projects

# Install dependencies
pip install pandas numpy matplotlib seaborn scikit-learn jupyter
```

### Running a Project

```bash
# Launch Jupyter Notebook
jupyter notebook

# Navigate to any project folder and open the .ipynb file
```
